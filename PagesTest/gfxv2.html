Fungsi tombol pada font editor:

Seluruh warna abu-abu adalah area tayang karakter.
ROWS    =   Menambah/Mengurangi jumlah baris pixel aktif
COLS    =   Menambah/Mengurangi jumlah kolom pixel aktif
BASE    =   Posisi keseluruhan pixel terhadap garis batas bawah tayang font (warna hijau ). Pengaruhnya di fungsi setCursor
XOFF    =   Posisi area tayang pixel karakter terhadap jarak antar karakter saat di print (garis merah kanan dan kiri)
XADV    =   Posisi batas akhir kolom pixel yang ditampilkan, dan berpengaruh ke jarak antar karakter(garis merah kanan)
DISABLE   =   Meng Aktif/Nonaktifkan karakter pada font
